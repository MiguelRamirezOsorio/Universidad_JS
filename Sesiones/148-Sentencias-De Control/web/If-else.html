Sentencia de Control If - Else:

Esta sentencia es del tipo condicional; Esta formada por la siguiente estructura:

if(condicion){

Ejecuta si la condicion es verdadera

}

else{

Ejecuta si la condicion es falsa

}

Donde si la condicion es verdadera, se ejecuta el primer corchete, y si la condicion es falsa, se ejecutara la sentencia else.
