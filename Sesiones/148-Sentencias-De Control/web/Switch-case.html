Sentencia de Control If - Else:

Esta sentencia es del tipo condicional; Esta formada por la siguiente estructura:

switch(variable){

case 1 :

Si se cumple opcion 1, se ejecuta

break;

case 2 :

Si se cumple opcion 2, se ejecuta

break;

:

:

case default :

Si no se cumple nada, se ejecuta

break;

}

Donde si la condicion en el case especifico es verdadero, se ejecuta el codigo que este en este hasta que suceda el break, y si la condicion del caso especifico es falsa, pasa a la siguiente condicion, asi sucesivamente.
Si no se cumple ningun caso especifico se ejecuta el caso default; acabando asi con la sentencia switch - case.
