Arreglos en JS:

Definición:

Los arrays son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación.
Tanto la longitud como el tipo de los elementos de un array son variables. Dado que la longitud de un array puede cambiar en cualquier momento, y los datos se pueden almacenar en ubicaciones no contiguas, no hay garantía de que los arrays de JavaScript sean densos; esto depende de cómo el programador elija usarlos.
En general estas características son cómodas, pero si, en su caso particular, no resultan deseables, puede considerar el uso de arrays con tipo.

Recorrido de Arreglos:

Para acceder a los elementos de un arreglo se debe saber que su primer elemento es el numero 0 y de ahi va en incremento. Tras saber esto, para acceder a un elemento del arreglo se hará uso la siguiente digitación:

variable[numero del elemento].

Para el recorrido general del arreglo se hara uso de las sentencias ciclicas como lo son el for o el while, esto quedará detallado de igual manera en el ejemplo a continuación:

Modificar elementos de arreglos:

Para cambiar un elemento especifico del arreglo se hará uso de la siguiente sentencia:

variable[numeroElemento] = cambio de elemento;

Agregar elementos de arreglos:

Para agregar un elemento en un arreglo se puede hacer de 3 formas distintas:

  • Uso del metodo push: Se hace uso de la sentencia:
    variable.push(nuevoElemento);
  • Uso del largo del arreglo: Se hace uso de la siguiente sentencia:
    variable[variable.length] = nuevoElemento;
  • Mediante los indices: Este puede dejar lugares vacios, se usa la siguiente sentencia: variable[numeroElemento] = nuevoElemento;

OJO:

Preguntar si es un array:

A partir de la version ECMA-SCRIPT 2015 se usa el siguiente método:

Array.isArray(variable);

Otra forma de saber si es un arreglo es mediante el instanceof:

variable instanceof Array;
