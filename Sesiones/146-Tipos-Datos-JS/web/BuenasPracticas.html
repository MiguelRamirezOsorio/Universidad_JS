Uso de camel case:

Se usa cuando el nombre de una variable excede las dos palabras; es asi que por ejemplo si el nombre de la variable es let apellidopaterno, mediante el camel case pasaria a ser: let apellidoPaterno. El ejemplo queda dado en la siguiente imagen:

Variables en una misma linea:

Para las variables en una misma linea se separará por comas cada variable, ademas de darle un espaciado en cada coma, por ultimo a cada variable que se encuentre en la misma linea tambien se le puede asignar en ese mismo lugar.

Sensibilidad entre mayusculas y minusculas:

En JavaScript tiene importancia el tipo de letra con el cual se nombran las variables debido a esto, la variable nombreCompleto no será igual a la variable nombrecompleto. Esto por esto que debes tener cuidado al definir tus varaibles en JavaScript.

Correcta defincion de nombres de variables:

El nombre de la variable cambiará de acuerdo las reglas especificadas:

  • Empezar con un numero.
  • Empexar con guion bajo o dolar.
  • Uso de palabras reservadas.
  • Modificacion de las palabras reservadas.
