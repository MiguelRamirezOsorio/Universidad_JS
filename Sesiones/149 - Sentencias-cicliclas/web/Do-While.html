Ciclo Do - While:

Esta sentencia es del tipo ciclica; se diferencia de la sentencia ciclica while en que esta ejecutara una vez mas el codigo a repetir en el momento en que la condicion por primera vez no sea verdadera. Esta formada por la siguiente estructura:

do{

Ejecuta mientras la condicion sea verdadera

}

while(condicion);

Etiquetas (labels):

Proporciona una sentencia con un identificador al que se puede referir al usar las sentencias break o continue.
Por ejemplo, puede usar una etiqueta para identificar un bucle, y entonces usar las sentencias break o continue para indicar si un programa debería interrumpir el bucle o continuar su ejecución.
